---
description: "Challenge assumptions and rebuild from irreducible truths. Usage: /consider:first-principles [topic]"
---

Apply first-principles thinking to: $ARGUMENTS (or the current discussion if no topic given).

<objective>
Decompose the topic into fundamental truths, challenge every assumption, and rebuild understanding from scratch. Block reasoning by analogy ("X worked for Y so...").
</objective>

<process>
1. State the problem or decision clearly in one sentence
2. List ALL assumptions being made — even "obvious" ones
3. Challenge each assumption: "Is this actually true, or just convention/habit/analogy?"
4. Identify the irreducible truths — what remains after challenges
5. Rebuild the solution using only those truths
6. Compare: does the rebuilt solution differ from the original approach?
</process>

<output_format>
**Problem:** [one sentence]

**Assumptions:**
- [assumption]: challenged — [true / false / partially true — why]
- ...

**Irreducible Truths:**
- [truth]: [why this can't be broken down further]
- ...

**Rebuilt Solution:**
[What you'd build starting from only the truths above]

**Delta:**
[How this differs from the current/obvious approach, and whether the difference matters]
</output_format>

<success_criteria>
- Every assumption is explicitly listed, not just the controversial ones
- No reasoning by analogy ("this is like..." or "X did it this way")
- The rebuilt solution could differ from the original — if it doesn't, explain why the original was already first-principles
- Actionable: the output leads to a decision or next step
</success_criteria>
