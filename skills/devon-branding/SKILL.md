# Devon Branding

Use this skill for Devon-branded UI, theme, presentation, document, or design
review work. Keep the entrypoint lean: read only the relevant sections from
[references/devon-brand-standards.md](references/devon-brand-standards.md) for
the surface you are changing.

## Composes With

- Parent: direct Devon branding or brand-compliance request.
- Children: none.
- Uses format from: none.
- Reads state from: requested design files plus [references/devon-brand-standards.md](references/devon-brand-standards.md).
- Writes through: requested UI, theme, token, document, presentation, or review artifact only.
- Hands off to: none.
- Receives back from: none.

## When To Use

- Devon-branded UI components, CSS, SCSS, Tailwind, design tokens, or themes
- dashboards or data visualizations that must use Devon colors
- slide decks, docs, or communications that need Devon voice and typography
- design or code review for Devon brand compliance

## Workflow

1. Identify the output surface: web UI, data visualization, presentation,
   document, iconography, logo placement, or photography direction.
2. Read only the relevant sections in
   [references/devon-brand-standards.md](references/devon-brand-standards.md):
   - color or token work: Color Palette, CSS Variables, Tailwind Config
   - typography work: Typography, Quick Reference
   - visual review: Logo Guidelines, Accessibility, Legal
   - image selection: Photography Guidelines
   - presentation or document work: Brand Voice, Typography, Quick Reference
3. Apply primary palette first. Use secondary colors in a supporting role only.
4. Prefer Tusker Grotesque for headlines and Aeonik for body copy. Use the
   documented fallback stacks only when the primary fonts are unavailable.
5. Preserve logo rules: full signature by default, 1x clear space, no
   distortion, no recoloring, and no wordmark-only usage.
6. Keep tone clear, purposeful, understated, and free of buzzwords.
7. Check WCAG contrast and call out missing fonts or official assets instead of
   inventing replacements.

## Non-Negotiables

- Primary accent: Clay Orange `#F04123`.
- Default body copy: Slate Gray `#4B4F54`.
- Headings: Iron Black `#212121`.
- Use brand colors at 100%; tints are allowed only for data visualizations.
- Do not redraw or improvise Devon logos. Use official assets from the Brand
  Resource Center when a logo or Strata Bars icon is needed.
- When trademark text is required, use `Devon Energy®`.
- When a footer or legal line is required, use the copyright format from
  [references/devon-brand-standards.md](references/devon-brand-standards.md).
